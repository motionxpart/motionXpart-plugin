
# About motionXpart plugin

motionXpart is a free, all-in-one utility panel for Adobe After Effects, designed to make everyday motion design and video editing workflows faster, cleaner, and smarter.

Created by Mohammad Hridoy—a full-time Motion Graphics Designer and Video Editor working in the industry since 2021—this extension was built out of real-world necessity. It compiles essential tools into a single, intuitive panel to eliminate repetitive tasks, automate timeline organization, streamline keyframe easing, and bridge the gap between client requirements and final render.

Whether you are organizing complex comps, automating subtitles with CaptionX, or customizing keyframe curves, motionXpart keeps you in your creative flow.
# 1. Quick Utilities

A collection of rapid alignment and timing tools to keep your timeline tidy.

## ✨ Features

### Interactive Anchor Point Grid
Instantly align the anchor points of selected layers to 9 standard positions:

- Top-Left
- Top-Center
- Top-Right
- Middle-Left
- Middle-Center
- Middle-Right
- Bottom-Left
- Bottom-Center
- Bottom-Right

Includes smart coordinate shifting to prevent layers from jumping on screen.

### Align to CTI (Playhead)
Quickly shift the in-point of selected layers directly to the Current Time Indicator.

### Advanced Layer Sequencing
Arrange layers sequentially with:

- Custom Frame Offset (FR)
- Step Count (ST)

Sequencing modes:

- Top to Bottom
- Bottom to Top
- Randomized

## Installation

Install motionXpart via ZXP extension

```bash
  First install the zxp/uxp installer by aescripts extension, and just drag and drop the motionXpart .ZXP file onto the ZXP installer.
  Then it will be installed. After that open AE, go to the top window (in software) and then go to Extensions,
  you will see motionXpart there.
```
