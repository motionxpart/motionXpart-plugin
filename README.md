
# About motionXpart plugin

The motionXpart plugin is designed to make it easy to create quick layers, create compositions, work with necessary expressions, add new expressions whenever you want, and add animated captions with one click, using Adobe After Effects.

Those who do not know After Effects well can also easily use this plugin. They can easily create animated captions through .SRT files.

The purpose of creating motionXpart is to make our daily work fast, easy, and smooth!
## Features

- Quick anchor point change
- Sequence Layers
- Smart folder Organizer
- Quick Layers & Comps
- Caption animation via .SRT file
- Quick edit captions
- Custom color button
- Tutorial video on how to use motionXpart


## Installation

Install motionXpart via ZXP extension

```bash
  First install the ZXP installer extension, and just drag and drop the motionXpart .ZXP file onto the ZXP installer.
  Then it will be installed. After that open AE, go to the top window (in software) and then go to Extensions,
  you will see motionXpart there.
```
